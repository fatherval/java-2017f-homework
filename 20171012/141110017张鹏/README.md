这次作业的对象总的来看一个是空间对象，还有一个是有机物对象。<br>

空间对应一个二维平面，每一格有他的坐标以及上面的物体。当然物体不拘泥于葫芦娃或者妖怪，也可以是土地，草丛或者河流。这次的作业中默认的构造是土地，在打印中表现为****。<br>
它的方法有：<br>
1.设置空间大小，这次将葫芦山设置为12*12。默认构造函数为7*7；<br>
2.植入子空间，将总空间的某个部分修改成另一个给定的空间，为直接植入排列好的生物阵法带来便利。后期也可以直接添加山水；<br>
3.打印该空间。<br>

空间取一个子类叫做阵法。它是一个二维平面，其中的物体要按照某种法则来排列。<br>
它的方法有：<br>
1.放置或读取领队，阵法中的领队往往是队伍中的老大，设置一个独立的方法是有必要的；<br>
2.设置或读取小弟；<br>
3.第三部分应该编写阵法的变换，但是这一次作业只需要打印出对峙场面，变换的过程是不要的。所以我们省略了这部分内容。在实现时直接以一个同样大小的另一阵法子空间覆盖之.<br>

根据不同的阵法要求，我们有各种子类。这次作业主要编写了长蛇，偃月，雁形以及鹤翼这四种。一来足够满足题目的要求，二来其余的阵法要么认不得，要么不好写。<br>

对于有机物对象，在这次的作业中没有采用接口，仍然选择了类。主要是看重这次的作业主体还都是带名字的生物，尚未出现地形等。那么使用带有成员属性（名字）的类是较为方便的。<br>
关于生物体的各种属性以及方法就不再叙述了。只是在助威方法中，所打印出来的“胜钧欧巴“是我的舍友，如有必要，修改成葫芦欧巴或者蝎子欧巴即可。<br>
